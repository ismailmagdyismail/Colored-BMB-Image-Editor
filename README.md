# Colored-BMB-Image-Editor

a program that manipulates colored bitmap images of size 256x256 
 Filter Available:
  1-Black & White Filter
  2-Invert Filter
  3-Flip Image
  4-Rotate Image
  5-Mirror Image
  6-Shuffle Image
  7-Blur Image
  8-Enlarge Image
  9-detect Edges
  10-Darken OR lighten Image
  11-Merge With another Image
  12-Shrink Image
